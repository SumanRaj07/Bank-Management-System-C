
# Bank Management System (C)

A simple console-based Bank Management System developed using C programming.
This project demonstrates the use of file handling to store bank account
details permanently and allows basic banking operations.

##  Project Description
The Bank Management System allows users to create a bank account, deposit
money, withdraw money, and view account details. All data is stored in a
file so that records are not lost when the program is closed.

##  Features
- Create new bank account
- Deposit money
- Withdraw money
- Display all account details
- File handling for permanent storage

##  Technologies Used
- C Programming Language
- Structures
- Loops and Switch Case

##  How to Run the Project
1. Clone or download the repository
2. Open `bank.c` in Dev-C++ or Code::Blocks
3. Compile and run the program
